7.StringAPI

Java重点String类包含50多种方法，大多数用的频率都很高

每一个API的注释都以类名开始，如java.lang.String，类名后是一个或多个方法的名字，解释和参数描述

（java.lang包名的重要性后面再说）

  1. 简洁列出常用方法

    1. char charAt(int index)

      返回指定位置的代码段元，返回指定字符串制定下标的char字符

    2. int codePointAt(int index)

      返回指定位置的码点

    3. int offsetByCodePoints(int startIndex, int cpCount)

      从startIndex码点开始，cpCount个码点后的码点索引

    4. int compareTo(String other)

      按照字典顺序，两个字符串比大小，0下标元素一样就继续往后，如果全部一样返回0，如果字符串在other之前那就返回负数，之后返回一个正数，一般返回的数为第一个不同的字符元素的ASCII码差值

    5. IntStream codePoints()

      *将这个字符串的码点作为一个流返回，调用toArray将它们放在一个数组中

    6. new String(int[] codePoints, int offset, int count)

      用数组中从offset开始的count个码点构造一个新的字符串

    7. boolean empty()

    8. booleam blank()

      用来判断字符串是否为 空 或 空格 组成，如果为空，返回true

    9. boolean equals(Object other)

    10. boolean equalsIgnoreCase(String other)

      判断字符串是否相等，可忽略大小写，如果相等返回true

    11. bolean startWith(String prefix)

    12. bolean endsWith(String suffix)

      如果字符串以prefix开头或以suffix结尾，返回true

    13. int indexOf(String str)

    14. int indexOf(String str, int fromIndex)

    15. int indexOf(int cp)

    16. int indexOf(int cp, int fromIndex)

      返回与字符串str或码点cp匹配的第一个子串开始的位置。从索引0或fromIndex开始。如果不存在str则返回-1

    17. int lastIndexof(String str)

    18. int lastIndexof(String str, int fromIndex)

    19. int lastIndexof(int cp)

    20. int lastIndexof(int cp, int fromIndex)

      返回与字符串str或码点cp匹配的最后一个子串开始的位置。从字符串结尾或fromIndex开始。如果不存在str则返回-1

    21. int length()

      返回字符串代码单元的个数

    22. int codePointCount(int startIndex, inr startIndex)

      返回startIndex，startIndex之间的码点个数

    23. String replace(CharSequence oldString, CharSequence newString)

      返回一个新的字符串，用newString去替换oldString。可以用String或StringBuilder对象作为CharSequence的参数

    24. String substring(int beginIndex)

    25. String substring(int beginIndex, int endIndex)

      返回一个新的字符串，从beginIndex开始到结尾后或到endIndex

    26. String toLowerCase()

    27. String toUpCase()

      返回一个新的字符串，将原字符串大写变小写或小写变大写

    28. String trim()

    29. String strip()

      返回一个新字符串。这个字符串删除原始字符串头部和尾部小于等于U+0020的字符（trim）或者空格（strip）

    30. String join(CharSequence delimiter, charSequence... elements)

      返回一个 新的字符串，用给定的定界符连接所有元素

    31. String repeat(int count)

      返回一个字符串，将当前字符串重复count次

注意：CharSequence类型的参数，这是一种接口类型，所有字符串都属于这个接口

后面再具体说，现在只需要知道 当看到一个CharSequence形参（parameter）时，完全可以传入String类型的的实参（argument）

控制流程

注意看书中每一小节开始的引入的那段话

  1. 块作用域

    再开始深入学习控制流程前，要先了解block块的概念

  2. 条件语句

  3. 循环

    当条件为true时，while循环执行一条语句，也可以是一个语句块

    while (condition) statement

    切记，如果初始的condition值就为false，那么while循环一次也不执行

    因为while循环是在执行前先检查条件，因此循环体代码可能一次也不执行

    如果需要循环体最少执行一次可以使用do/while循环

    do {statement...} while (condition)

  4. 确定循环

    for循环语句是支持迭代的一种通用结构，有一个计数器或类似的变量控制迭代次数，每次迭代后这个变量会更新。

    for语句：

    • 第一部分：通常是对计数器进行初始化
    • 第二部分：给出每次新一轮循环执行前都要检测的循环条件
    • 第三部分：指定如何更新计数器

    后面说Java5新加的 “泛型for循环” 即 for each循环

  1. 多重选择：switch语句

    在处理多个选项时，使用if/else结构显得有些笨拙，这时Java提供了一个和C/C++完全一样的switch语句

    例如，建立一个包含四个选项的菜单系统

    switch语句将从与选项值相匹配的case开始执行，直到遇到break语句，或者执行到语句快结束，如果没有一个匹配的选项，有default的话就执行default语句；

    如果没有在case后写break，那么可能触发多个case

    判断好哪个case后如果这个case没有写break就会直接执行下一个case

    case标签可以是：

    - 类型为char，byte，short，或int的常量表达式子
- 枚举常量
- 从Java7开始，还可以是字符串字面量

  2. 中断控制流程的语句

    尽管java的设计者吧goto作为保留字，但实际上并没有打算在语言上使它。

    通常，goto语句被认为是一种拙劣的程序设计风格

    带标签的break语句，用于跳出多重循环嵌套的语句。

    continue跳过当前循环体的剩余部分，跳到最内层循环的首部

数组

数组即储存相同类型值的序列

  1. 声明数组

    int[] a; 支出数组类型（数组元素类型紧跟[]）和数组变量名字

    这只声明了一个整型数组，并没有将a初始化为一个真正的数组，应该使用new操作符创建数组

    int[] a = new int[100];

    这条语句声明并初始化了一个可以存储100个整数的数组。

    数组是一种数据结构，是用来存储同一类型值的集合。

    通过一个整形下标（index，或称作索引）可以访问数组中的每一个值

  2. 访问数组元素

    首先，前面创建的数组的下标为从0-99，不是1-100.

    可以使用一个循环去给数组填充元素

    创建一个数字数组后，所有元素都初始化为0

    boolean数组元素都初始化为false

    对象数组的元素都初始化为一个特殊值null，表示这些元素还未存放任何对象
